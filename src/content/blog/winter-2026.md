  <h2 style="margin: 0 0 0.95rem; color: #234e75; font-size: 1.75rem; font-weight: 700; line-height: 1.15;">
    Tax Planning for 2025 Filing
  </h2>
  <p style="margin: 0 0 0.95rem; color: #334155; font-size: 1rem; line-height: 1.8;">
    You can make the most of your hard-earned income during the first 60 days of the year with an RRSP contribution if you missed it during the year. The maximum allowed for this year's deduction is $32,490 with it being based on 18% of your earned 2025 plus any carryforward from previous years.
  </p>
  <p style="margin: 0 0 1rem; color: #334155; font-size: 1rem; line-height: 1.8;">
    You may be able to use the HATC (Home Accessibility Tax Credit) for either disabled people or seniors if you qualify. As usual, it is not straight forward as there is criteria to use this credit.
  </p>
  <ul style="margin: 0; padding: 0; list-style: none; color: #334155;">
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>Qualifying person must have the DTC (Disability Tax Credit) or be over 65 by Dec 2025</span>
    </li>
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>Eligible person must be either a spouse or a parent, grandparent, direct blood relative who claims the qualifying family member as an eligible dependent or is the Caregiver of that person with no income</span>
    </li>
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>Eligible dwelling must be the building where both people ordinarily live in this house</span>
    </li>
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>Eligible renovation expenses must be permanent &amp; integral to allow the "qualifying" person greater access, function or mobility that reduces risk of harm to them</span>
    </li>
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>The list of items that allowed are:</span>
    </li>
  </ul>
  <div style="margin: -0.2rem 0 0.9rem 1.85rem; padding: 0.2rem 0 0.2rem 1rem; border-left: 2px solid #d7e1eb;">
    <ul style="margin: 0; padding-left: 1rem; color: #334155; line-height: 1.75;">
      <li>Building materials</li>
      <li>Fixtures</li>
      <li>Equipment rental</li>
      <li>Building plans</li>
      <li>And permits</li>
    </ul>
  </div>
  <ul style="margin: 0; padding: 0; list-style: none; color: #334155;">
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0 0 0.8rem;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>NOTE: Labour &amp; tools do not qualify &amp; the person/firm selling them must charge GST/HST</span>
    </li>
    <li style="display: flex; gap: 0.65rem; align-items: flex-start; margin: 0;">
      <input type="checkbox" disabled style="margin-top: 0.28rem; flex: none;" />
      <span>You must keep all the receipt in case CRA requests proof of the expenses</span>
    </li>
  </ul>
  <p style="margin: 1rem 0 0; color: #334155; font-size: 1rem; line-height: 1.8;">
    If you can check all theses boxes off, you are able to use this credit to help reduce your taxable payable.
  </p>
